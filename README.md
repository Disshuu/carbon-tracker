## 🧩 Problem Statement

Most people have never calculated their carbon footprint.
The average Indian emits **1,900 kg CO₂/year** — but has no idea where it comes from.
Existing tools are complex, boring, and not personalized.

**EcoTrace fixes that in 60 seconds.**

---

## ✨ Features

| Feature | Description |
|---|---|
| 🧮 3-Step Calculator | Transport · Food · Home Energy |
| 📊 Live CO₂ Score | Updates in real-time as you fill in data |
| 📈 Visual Dashboard | Pie chart + benchmark vs India & Global avg |
| 🤖 AI-Powered Insights | Personalized tips based on YOUR actual data |
| 🇮🇳 India-Specific | Uses Indian grid factor (0.82 kg CO₂/kWh) & LPG cylinders |

## 🛠️ Tech Stack

- **Frontend:** React.js

---

## 📐 Emission Factors Used

| Category | Factor |
|---|---|
| Petrol car | 0.21 kg CO₂/km |
| Short-haul flight | 255 kg CO₂/flight |
| India electricity grid | 0.82 kg CO₂/kWh |
| LPG cylinder (14.2 kg) | 2.04 kg CO₂/cylinder |
| Diet (vegan → meat-heavy) | 1,500 – 3,300 kg CO₂/yr |

## 🚀 Run Locally

```bash
# Clone the repo
git clone https://github.com/Disshuu/carbon-tracker.git
cd carbon-tracker

# Install dependencies
npm install

# Start the app
npm start
```

App runs at `http://localhost:3000`

---

## ☁️ Deploy to Vercel

1. Push code to GitHub (public repo)
2. Go to [vercel.com](https://vercel.com) → Import your repo
3. Click **Deploy**
4. Done — live in 2 minutes!
